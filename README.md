# E-Commerce Customer Behavior Analysis

## Project Overview
This project analyzes a dataset of ~17,000 e-commerce transactions from a Turkish retail platform to identify growth opportunities and customer behavioral patterns.

## Business Questions to Answer
1. **Revenue & Seasonality:** Which months and days generate peak revenue?
2. **Customer Loyalty:** How does Lifetime Value (LTV) differ between returning vs. new customers?
3. **Engagement Correlation:** Do session duration and pages viewed correlate with spend?
4. **Operational Performance:** How does delivery time impact customer ratings?
5. **Discount Efficiency:** Do discounts drive volume or just hurt margins?

## Tech Stack
* Microsoft Excel (Pivot Tables, Power Query, Advanced Formulas)
